# Jarvis-ai-assistant
A Python-based AI voice assistant that can listen to voice commands, open websites, play music, fetch news, and interact using speech recognition and text-to-speech.
## Features
- Voice Recognition
- Text To Speech
- News Fetching
- Website Automation
- AI Chat Integration

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- gTTS
- OpenAI API
- Requests
